# 🪖 Controle de EPI Taboca

Sistema de Controle de Equipamentos de Proteção Individual (EPI) desenvolvido para a **Taboca**. 

## 🔗 Link do Aplicativo
Acesse o aplicativo publicado em:  
👉 **[https://eullon1234-creator.github.io/CONTROLE-DE-EPI-TABOCA-](https://eullon1234-creator.github.io/CONTROLE-DE-EPI-TABOCA-)**

---

## 💻 Instalação no Computador (PWA)
Este aplicativo é uma PWA (Progressive Web App) e pode ser instalado diretamente no seu computador ou celular para funcionar como um programa independente:
1. Abra o link do app no **Google Chrome** ou **Microsoft Edge**.
2. Clique no ícone de instalação (computador com uma seta para baixo) na barra de endereços do navegador.
3. Se preferir, clique no botão **💻 Baixar App no PC** no menu lateral do aplicativo.

---

## 🛠️ Principais Recursos
* **📊 Dashboard:** Gráficos e indicadores de estoque baixo e movimentações diárias.
* **📤 Registrar Saída:** Registro de baixas de EPI com histórico e controle de estoque automático.
* **📥 Registrar Entrada:** Registro de compras e reposição de saldo de estoque.
* **📦 Controle de Estoque:** Visualização de saldo atual, estoque mínimo/máximo e status de alerta.
* **🗂️ Cadastro de Produtos:** Gerenciamento de códigos, descrição, Certificado de Aprovação (CA), validade de CA, unidades de medida e localização física.
* **📍 Localização Física (Prateleiras):** Mapeamento físico dos EPIs (P1, P2, P3, P4) para agilidade no estoque.
* **🖨️ Ficha de Saída A4 (Impressão):** Impressão de ficha física simplificada com área de anotação manual de quantidades para posterior baixa digital.

## 🚀 Como executar localmente
1. Instale as dependências:
   ```bash
   npm install
   ```
2. Execute o servidor de desenvolvimento:
   ```bash
   npm run dev
   ```
3. Abra `http://localhost:5173/CONTROLE-DE-EPI-TABOCA-/` no navegador.
